# Event-Ticket-Booking-System-project

The Event Ticket Booking System is a desktop application that helps manage events 
and ticket bookings in a simple and organized way. It supports three main types of users: 
Customers, Event Managers, and Administrators. Each user has access to specific 
features depending on their role. 
 
Customers can browse all available events, view event details such as date, location, 
available seats, and price, and book the tickets they want. They can also cancel their 
bookings and review their booking history at any time, which makes the process of 
attending events easier and more convenient. 
 
Managers can create new events, edit existing ones, or delete events when necessary. 
They can also view the events they manage and check related statistics, allowing them to 
organize and update their events efficiently. 
 
Administrators have full access to the system. They can manage user accounts by 
creating new users or deleting existing ones, and they can view all events and all booking 
records in the system. They also have access to overall statistics, giving them a complete 
overview of the system’s performance.
